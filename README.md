## What This Is

The Nexus is not a database. It is a **structured knowledge superposition layer** — where entities, contexts, and relationships create emergent understanding through structured overlap.

Rather than passing conversations between AI systems, every participant communicates through a common protocol that provides a structured representation of reality.

## Clone-to-Node Protocol

```bash
# One-liner: clone this repo and become a mesh spoke
curl -sL https://raw.githubusercontent.com/q-u-i-l-l-y/quillyos-nexus/main/install/install.sh | bash
```

Anyone who clones this repository becomes a **spoke node** in the QuillyOS mesh. The repository is the protocol. The protocol is the mesh.

## Data Matrices Superimposition

The core insight: when knowledge layers overlap, trends emerge that no single layer contains.

| Layer | Name | Function |
|-------|------|----------|
| 0 | Observation | Raw sensor input, raw data capture |
| 1 | Evidence | Filtered, timestamped, attributed observations |
| 2 | Context | Evidence placed in architectural & temporal context |
| 3 | Translation | Context rendered into agent-specific formats |
| 4 | Consensus | Cross-agent validated knowledge |
